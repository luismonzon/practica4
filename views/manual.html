Descripcion

La aplicacion consiste en el ingreso y muestra de datos de desde una base de datos con PostgreSQL a traves de NodeJs Se podran realizar consultas acerca de rutas, viajes y buses del sistema Ademas se podran comprar y reservar tickets

  • Bootstrap 3.0
  • Body Parser
  • Pg plugin PostgreSQL for NodeJs
